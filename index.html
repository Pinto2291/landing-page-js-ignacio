Acerca

Como Ingeniero Civil y Desarrollador Web, tengo conocimientos en JavaScript, HTML, CSS, React, Node y bases de datos. Soy capaz de crear sitios web dinámicos y adaptativos a múltiples pantallas. HTML es el lenguaje de marcado que utilizo para estructurar el contenido de una página web, CSS para dar estilo y diseño, y JavaScript para añadir interactividad y dinamismo. React y Node son herramientas que utilizo para crear aplicaciones web más complejas y escalables. Además, tengo experiencia en el desarrollo de aplicaciones móviles y en la sincronización de datos.

Tecnologias

HTML

HTML (Hyper Text Markup Language) es un lenguaje de marcado de etiquetas que se utiliza para crear y estructurar contenido en la web. Este lenguaje permite definir la estructura y el contenido de una página web mediante el uso de etiquetas que indican al navegador cómo mostrar el contenido de la página.

CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para definir la presentación de un documento HTML. CSS permite separar el contenido de un documento HTML de su presentación, lo que facilita la creación de diseños y estilos coherentes en toda una página web.

JavaScript

JavaScript es un lenguaje de programación interpretado que se utiliza principalmente para crear interactividad dinámica en los sitios web. Es un lenguaje de alto nivel, dinámico e interpretado que se define como orientado a objetos, basado en prototipos, imperativo y débilmente tipado.

Herramientas

VS Code

Visual Studio Code (VS Code) es un editor de código fuente gratuito y de código abierto desarrollado por Microsoft. VS Code es altamente personalizable y cuenta con una amplia variedad de extensiones y herramientas para desarrolladores. Es compatible con múltiples lenguajes de programación, incluyendo JavaScript, Python, C#, HTML y CSS.

Figma

Figma es una herramienta de diseño de interfaces de usuario que se utiliza para crear y diseñar prototipos de aplicaciones y sitios web. Figma es una aplicación basada en la nube que permite a los usuarios trabajar en equipo y colaborar en tiempo real.
